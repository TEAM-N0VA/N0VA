# N0VA
이 레포지토리는 2025 가을학기 캡스톤디자인과창업프로젝트 '스타트' 과목에서 22팀 NOVA의 구독 사용 패턴 탐지 및 graph RAG 기반 구독 효율 최적화 서비스 구현을 목표로 합니다. 

## 구독 서비스의 자동 결제로 인해 지출을 줄이고 싶지만 방치하는 사용자를 위한 구독 사용 패턴 탐지 및 RAG 기반 개인 맞춤형 구독 효율 최적화 및  추천 서비스
사용자의 결제 내역과 앱 사용량을 연동하여 방치된 구독을 분석하고, 매주 실질적인 가성비 리포트를 제공하여 불필요한 지출을 방지합니다. 또한 복잡한 약관, 통신사/카드 혜택 등을 분석하여 개인에게 최적화된 요금제 조합을 추천할 예정입니다.

## 프로젝트 개요
구독 경제의 확산으로 결제는 간편해졌지만, 파편화된 결제 주기와 복잡한 혜택 정보로 인해 구독 피로도와 비효율적 지출이 늘어나고 있습니다. 본 프로젝트는 단순히 구독 목록을 나열하는 가계부를 넘어, 사용자가 실제로 서비스를 얼마나 활용하는지를 분석하여 효율성을 진단합니다. 또한, Neo4j 기반의 지식 그래프 기술을 도입하여, 복합적인 질문에 대해 정확한 솔루션을 제공합니다.

### 목표
- 결제 내역과 실제 앱 사용량 분석을 통한 실질적인 효율을 진단하고, 좀비구독(사용하지 않는데 결제되는 서비스) 문제 해결
- 흩어진 카드사/통신사 제휴 혜택을 통합 분석하여 최적의 할인 조합 추천
- 방치된 서비스의 사용 독려 또는 합리적인 대체 플랜/서비스 추천을 통한 행동 가이드 제공
  
### 주요기능
#### 1. 구독 통합 관리 및 분석 : 흩어진 구독 내역, 결제 금액, 결제 정보의 통합 조회 및 관리
- 사용자별 소비 패턴을 분석하여 3단계 등급(Light / Normal / Heavy) 산출 및 이상치 탐지

#### 2. 주간 가성비 리포트: 매주 사용량 기반의 효율 리포트 발행 및 맞춤형 피드백 제공
#### 3. 구독 혜택 자동 조합 및 최적화
- 사용자의 보유 카드, 통신사 정보, 월 결제 패턴을 종합하여 실질 체감가가 가장 낮은 최적의 혜택 조합 추천

#### 3. RAG 기반 맞춤 상담 챗봇: 구독 정책, 약관, 제휴 혜택(카드/통신사) 등 방대한 데이터를 기반으로 개인 최적화 플랜 상담 및 사용자 의도(최저가 조합 요청, 해지 방법 문의 등)에 따른 솔루션 제공
- 복잡하게 얽힌 상품-요금제-혜택 관계를 그래프로 연결하여, 단순 검색으로는 불가능한 맥락 기반 답변을 제공

### 서비스 기대효과 
* 좀비 구독 차단 및 최적 혜택 적용을 통해 월 구독 지출 10~30% 절감.

* 플랫폼별로 분산된 약관 확인 및 해지 절차 탐색 시간 단축.

* 데이터 시각화를 통해 자신의 소비 생애 주기를 주도적으로 관리.
