# N0VA
이 레포지토리는 2025 가을학기 캡스톤디자인과창업프로젝트 '스타트' 과목에서 22팀 NOVA의 구독 사용 패턴 탐지 및 RAG 기반 구독 효율 최적화 서비스 구현을 목표로 합니다. 사용자의 결제 내역과 앱 사용량을 연동하여 방치된 구독을 분석하고, 매주 실질적인 가성비 리포트를 제공하여 불필요한 지출을 방지합다. 또한 RAG 기술을 활용해 복잡한 약관, 통신사/카드 혜택 등을 분석하여 개인에게 최적화된 요금제 조합을 추천하고, 패밀리/그룹 플랜 매칭을 통해 고정 지출을 효율적으로 관리하도록 돕는 데 목적이 있습니다.
## 구독 서비스의 자동 결제로 인해 지출을 줄이고 싶지만 방치하는 사용자를 위한 구독 사용 패턴 탐지 및 RAG 기반 개인 맞춤형 구독 효율 최적화 및  추천 서비스
본 서비시는 관성적인 자동 결제로 방치되는 구독 서비스를 관리하고, 개인 맞춤형 플랜을 제안하는 구독 효율 최적화 서비스이다. 사용자의 구독 패턴을 분석해 매주 가성비 리포트를 발행하며, RAG 기반 챗봇을 통해 최저가 조합, 해지 방법, 대체 플랜 등 복잡한 구독 고민을 해결해 주는 개인 비서 역할을 수행합니다.

## 프로젝트 개요
### 목적
- 복잡한 구독 요금제와 관성적 자동 결제로 인한 불필요한 지출 방지 및 경제적 구독 방식 제안

- 결제 내역과 실제 앱 사용량 분석을 통한 실질적인 효율 진단

- 방치된 서비스의 사용 독려 또는 합리적인 대체 플랜/서비스 추천을 통한 구독 리모델링
### 주요기능
#### 1. 구독 통합 관리 및 분석 : 흩어진 구독 내역, 결제 금액, 결제 정보의 통합 조회 및 관리

#### 2. 주간 가성비 리포트: 매주 사용량 기반의 효율 리포트 발행 및 맞춤형 피드백 제공

#### 3. RAG 기반 맞춤 상담 챗봇: 구독 정책, 약관, 제휴 혜택(카드/통신사) 등 방대한 데이터를 기반으로 개인 최적화 플랜 상담 및 사용자 의도(최저가 조합 요청, 보고 싶은 작품 검색, 해지 방법 문의 등)에 따른 솔루션 제공

#### 4. 스마트 매칭 커뮤니티: 패밀리/그룹 플랜 매칭을 지원하여 고정 지출 절감 지원
