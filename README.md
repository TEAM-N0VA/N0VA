# 밀당(Meal당)
지속적인 혈당 관리가 어려운 임신성 당뇨 산모를 위한 YOLO 활용한 식단 자동 기록 및 RAG 기반 식단 코칭 서비스

> 밀당은 임신성 당뇨 산모를 위한 맞춤형 혈당 관리 모바일 애플리케이션입니다. 음식 사진을 한 장 찍으면 음식 종류와 영양 성분이 자동으로 기록되고, 이 식사를 했을 때 혈당이 얼마나 오를지 미리 예측해줍니다. 또한 산모의 혈당 패턴과 임신 주수에 맞는 식단을 추천하고, 외식 시에는 주변에서 안심하고 먹을 수 있는 식당과 메뉴를 안내합니다. 임신성 당뇨에 특화된 챗봇을 통해 궁금한 점도 언제든 상담받을 수 있습니다. 이를 통해 임신성 당뇨 산모가 매 끼니마다 느끼는 불안감과 식단 관리의 부담을 줄이고, 일상 속에서 편안하게 혈당을 관리할 수 있는 환경을 제공합니다.
